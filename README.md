# Random Task Generator

**Автор:** Меркулов Фёдор

## Описание

Графическое приложение на Python, позволяющее генерировать случайные задачи из предопределённого или пользовательского пула с разбивкой по категориям: **учёба**, **спорт**, **работа**.  
Поддерживает добавление новых задач, фильтрацию по типу при генерации и просмотре истории.  
Вся история автоматически сохраняется в JSON-файл и загружается при следующем запуске.

## Основные возможности

- Генерация случайной задачи с учётом заданного фильтра по типу.
- Добавление пользовательских задач (с проверкой на пустую строку).
- Отображение полной истории сгенерированных задач с возможностью фильтрации.
- Сохранение истории в `tasks.json` и её автоматическая загрузка.
- Простой и интуитивный интерфейс на Tkinter.
